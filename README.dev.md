# Toplumsal Fren Sistemleri — Geliştirme README

Bu çalışma depo sahibinin README'sine dayanılarak hızlı bir Python iskeleti ve örnek simülasyon eklenmiştir.

Hızlı başlangıç:

- Paket: `toplumsal_fren_sistemleri`
- Örnek çalıştırma: `python run_simulation.py`
- Testler: `python -m unittest discover -v`

Ne eklendi:
- `toplumsal_fren_sistemleri/core.py` — Basit sosyal fren simülasyonu.
- `run_simulation.py` — Kısa örnek çalıştırıcı.
- `tests/test_core.py` — Birkaç birim testi.

Geliştirme notları:
- Bu iskelet genişletilip daha gerçekçi araç modelleri, görselleştirme ve veri kaydı eklenebilir.
